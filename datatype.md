# 자료형 시작하기

간단하게 string, number, boolean, null, undefined, array, object, function, class를 중심으로 다룰 수 있습니다.
이정도는 자바스크립트를 작성하면서 외워지게 됩니다.

```JavaScript
string = 'hello, JavaScript!';
```

```JavaScript
number = 100;
```

숫자입니다. 정수랑 소수가 존재하는데


자료형을 조금 더 깊게 이해해보면 원시형(primitive)과 참조형이 있습니다.

자바스크립트의 원시형 자료형입니다.

string

number

bigint

boolean

null

undefined

symbol


array

object
